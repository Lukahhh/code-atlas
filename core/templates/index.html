What is Django Notebook?

Django Notebook was created in spite of wasting so much time googling trival things I knew I had done before. I often found myself looking through previous projects for code extracts. I thought there had to be an better way...
  1. 1

    Create a Note

    Notes are for anything that was hard to find and even harder to remember. If you think it might be useful in the future, add it to a note.

  2. 2

    Search your Notes

    You're stuck. You can't remember how you solved that tricky problem you spent hours on a few months ago. Oh wait! By searching your Notes, you easily find what you did last time!

  3. 3

    Edit and Improve

    Edit your Notes at any time, add references as well as get direct results from the Django documentation.

Why use Django Notebook?

It can be tricky to keep your notes in one place. As soon as you lose discipline, you're in the same situation of scattered notes all over again. With Django Notebook, everything is in one place. Made for making your life easier, features include:

Markdown Editing
Tagging
Url References
Quick Search
Documentation Search
